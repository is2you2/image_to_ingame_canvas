# 준비된 이미지를 인게임 팔레트로 전환 처리
png 와 같은 파일로 준비된 이미지를 인게임 캔버스로 옮기는 작업을 명령어로 수행하기

## 작업 단계

1. 팔레트 설정
- 사용 가능한 색상 리스트를 설정합니다. 색상을 선택할 때 필요한 명령을 같이 구성합니다
2. 캔버스 설정
- 인게임 캔버스의 위치 및 해상도, 크기 정보를 입력합니다
3. 이미지 설정
- 캔버스 설정에 기반하여 이미지 초안을 구성합니다
4. 팔레트 리맵핑
- 팔레트 설정파일과 이미지 초안을 불러와 이미지 색상을 재분배합니다
5. 명령 구성
- 모든 정보를 비교하여 명령어를 구성합니다
